# Automa-o-de-Gmail-com-Banco-de-Dados-da-empresa.
RPA com Python: Automação completa de relatórios. O script acessa o Google Drive via PyAutoGUI, baixa bases em Excel, processa indicadores de faturamento com Pandas e envia o relatório formatado automaticamente por Gmail. Foco em eficiência operacional e redução de erro humano, transformando horas de trabalho em minutos.
